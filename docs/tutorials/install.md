# Install

Per-platform install paths. The [Quickstart](../quickstart.md) covers the common case; this page is for the platform-specific edge cases.

This page is a placeholder for SPEC-38.5. Substantial content lands in a follow-up batch.

For now: `pipx install efterlev` works on macOS arm64/x86_64, Linux x86_64/arm64, and Windows. The container at `ghcr.io/efterlev/efterlev:latest` works on every Docker-capable host. The [`[bedrock]` extra](../tutorials/deploy-govcloud-ec2.md) opts in to the AWS Bedrock backend for GovCloud deployment.
